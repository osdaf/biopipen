    </>

    Metabolic landscape analysis for scRNA-seq data

    An abstract from https://github.com/LocasaleLab/Single-Cell-Metabolic-Landscape

    If you have clustering done somewhere else, you could use replace_clustering()

    See docs here

    https://pwwang.github.io/biopipen/pipelines/scrna_metabolic

    Reference: [1] Xiao, Zhengtao, Ziwei Dai, and Jason W. Locasale. "Metabolic landscape of the tumor microenvironment at single cell resolution." Nature communications 10.1 (2019): 1-12.

    Start Process: MetabolicInputs

    Functions
    • build_processes(options) — Build processes for metabolic landscape analysis pipeline</>
    • metabolic_landscape() (Pipen) — Build a pipeline for pipen run to run</>
